

# Obsidian 快捷键参考

## 基础操作
| 快捷键 | 备注 |
| ------------ | ---- |
| ctrl/cmd + N | 新建笔记 |
| ctrl/cmd + O | 快速打开（可能冲突，建议修改） |
| ctrl/cmd + P | 打开命令面板 |
| ctrl/cmd + , | 打开设置 |
| ctrl/cmd + W | 关闭当前标签页 |

## 编辑操作
| 快捷键 | 备注 |
| ------------ | ---- |
| ctrl/cmd + B | 加粗 **text** |
| ctrl/cmd + I | 斜体 *text* |
| ctrl/cmd + K | 插入链接 [[note]] |
| ctrl/cmd + ] | 增加缩进（列表/块） |
| ctrl/cmd + [ | 减少缩进 |
| ctrl/cmd + shift + ` | 建议：添加代码块快捷键（默认无） |

## 导航操作
| 快捷键 | 备注 |
| ------------ | ---- |
| ctrl/cmd + ←/→ | 切换标签页（可能与系统冲突） |
| ctrl/cmd + Tab | 切换最近面板 |
| ctrl/cmd + shift + L | 切换侧边栏（默认无，建议添加） |

## 视图操作
| 快捷键 | 备注 |
| ------------ | ---- |
| ctrl/cmd + E | 切换编辑/预览模式 |
| ctrl/cmd + shift + E | 切换源码模式 |
| ctrl/cmd + shift + 1/2/3 | 切换布局（单栏/双栏/三栏） |

## 搜索操作
| 快捷键 | 备注 |
| ------------ | ---- |
| ctrl/cmd + F | 当前文件搜索 |
| ctrl/cmd + shift + F | 全局搜索 |
| ctrl/cmd + G | 查找下一个（搜索后使用） |

## 建议修改

1. **快速切换面板** → 建议设置为 `ctrl+``
2. **代码块** → 建议设置为 `ctrl+shift+``
3. **侧边栏切换** → 建议改为 `ctrl+shift+L`

> 修改方法：通过 `ctrl + ,` 打开设置 → 快捷键 → 搜索功能修改
